# Traffic-Sign-Detection
Traffic Signal Detection using CNN to classify Red, Yellow, and Green signals from images.

##  Project Overview

Traffic signal detection is an important application in autonomous driving and intelligent transportation systems. This project focuses on detecting and classifying traffic signals from images using deep learning techniques.

---

##  Objectives

* Detect traffic signals from images
* Classify signals (Red, Yellow, Green)
* Process real-world images
* Build an accurate deep learning model

---

##  Dataset Description

The dataset contains labeled images of traffic signals:

* Red Signal
* Yellow Signal
* Green Signal

Includes real-world variations like lighting, angles, and backgrounds.

---

##  Data Preprocessing

* Resize images
* Normalize pixel values
* Convert images to arrays
* Split into training and testing data

---

##  Exploratory Data Analysis

* Visualized sample images
* Checked class distribution
* Observed variations

---

##  Model Used

Convolutional Neural Network (CNN)

Why CNN?

* Best for image data
* Automatic feature extraction
* High accuracy
